Projeto desenvolvido para fins de estudo e portfólio, inspirado na experiência da Netflix. O objetivo foi aplicar conceitos modernos de desenvolvimento Front-end utilizando HTML, CSS e JavaScript puro, simulando funcionalidades reais de uma plataforma de streaming.

## 🚀 Demonstração

🔗 Projeto Online:
https://ileekhan.github.io/SiteLoginNetflix/

## 📋 Funcionalidades

* Sistema de Cadastro e Login
* Seleção de Perfis
* Banner Dinâmico Automático
* Catálogo de Filmes e Séries
* Busca em Tempo Real
* Favoritos (Minha Lista)
* Histórico de Assistidos
* Modal de Informações
* Reprodução de Vídeos
* Layout Responsivo
* Persistência de Dados com LocalStorage

## 🛠 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* LocalStorage
* Git
* GitHub
